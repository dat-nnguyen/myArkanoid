# myArkanoid

A classic brick-breaker game inspired by the arcade legend Arkanoid (1986).  
Developed in **Java** with **JavaFX** on macOS, the project showcases Object-Oriented Programming, game loop architecture, collision detection, scoring system and level design.

## 🎮 Features  
- Paddle control to bounce a ball and break bricks.  
- Various obstacle types and levels that increase in difficulty.  
- Score tracking and restart functionality.  
- Clean modular code architecture: classes for `Paddle`, `Ball`, `Brick`, `LevelManager`, `ScoreManager`, `SoundManager`.    
- Kept frame rate smooth (60 FPS) and responsive input handling.

## 🛠 Tech Stack  
**Languages:** Java

**Libraries / Frameworks:** JavaFX

**Development Platform:** macOS

**Tools:** Git,IntelliJ 

🧠 Design Decisions & Learning Outcomes
- Used OOP to decouple game object logic, making the codebase modular and maintainable.

- Implemented efficient collision detection and response for bricks, walls, and paddle.

- Managed audio (music, sound effects) through a dedicated SoundManager.

- Designed a scoring system and UI overlay to display high scores.

- Enhanced my understanding of real-time game loops, delta-time handling, event polling, and resource management.

🧭 Roadmap & Future Enhancements
- Add “Power-up” items (multi-ball, paddle enlarge, laser) inspired by the original Arkanoid.

- Include multiple levels with unique brick layouts and increasing difficulty.
